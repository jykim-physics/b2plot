Plotting Style Guide
====================





**This is a brainstorming towards a publication guide for plots**

Having a common plotting style within the collaboration is a welcome goal.
As there is a lot of room for discussion, lets start with the most important guidelines.


**Disclaimer**

*At the moment this is just a personal proposal.*


**General Remarks**


The most important goal of a plot is to transport information visually,
the easier it is to read, the better.


*The most important points are:*

* Large label size
* Distributions easy to distinguish by color *and* surface
* Black and white readability
* Colorblind save
